Belegungsplan und Gästedatenbank Rezervi

Bitte beachten Sie die Installationsanweisungen.Sie finden diese in der Datei /doc/install.html.

Insbesondere muss; als erstes die Konfigurationsdatei angepasst werden. öffnen sie die Datei "rdbmsConfig.php" im Verzeichnis "conf" mit ihrem bevorzugten Editor und tragen sie die dort angegebenen Attribute ein.

Ein Installationsscript finden sie im Verzeichnis "install":

install/install.php

Nach erfolgreicher Installation können sie Rezervi über folgende URLs verwenden:

http://www.ihre-domain.com/IhrInstallationsverzeichnis/start.php

Dieser Startseite können Sie auch die gewünschte Sprache übergeben:

http://www.ihre-domain.com/IhrInstallationsverzeichnis/start.php?spracheNeu=de
http://www.ihre-domain.com/IhrInstallationsverzeichnis/start.php?spracheNeu=en
http://www.ihre-domain.com/IhrInstallationsverzeichnis/start.php?spracheNeu=fr

Den Zugang zur Webschnittstelle erhalten sie über:
http://www.ihre-domain.com/IhrInstallationsverzeichnis/webinterface/index.php

Bei Fragen können sie sich gerne jederzeit an uns wenden:

rezervi@utilo.net

http://www.utilo.net

Danke für ihr Interesse an Rezervi!
