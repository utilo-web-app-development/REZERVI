Rezervi Availability Overview and Guest Database

Please read the installation instructions. You find it in the file /doc/install_en.html.

First you have to do the configuration of Rezervi in the file "rdbmsConfig.php" in folder "conf" with your favorite editor.

After the installation you can use Rezervi with the following URLs:

http://www.your-domain.com/yourInstallFolder/start.php

You can specify your favorite language on this URL:

http://www.your-domain.com/yourInstallFolder/start.php?spracheNeu=de
http://www.your-domain.com/yourInstallFolder/start.php?spracheNeu=en
http://www.your-domain.com/yourInstallFolder/start.php?spracheNeu=fr

Your access the the Webinterface is with the URL:
http://www.your-domain.com/yourInstallFolder/webinterface/index.php

Please kontakt us, if you have some questions:

office@utilo.eu

http://www.utilo.eu

Thank you for your interest on Rezervi!

Belegungsplan und Gästedatenbank Rezervi

Bitte beachten Sie die Installationsanweisungen.Sie finden diese in der Datei /doc/install_de.html.

Insbesondere muß als erstes die Konfigurationsdatei angepasst werden. Öffnen sie die Datei "rdbmsConfig.php" im Verzeichnis "conf" mit ihrem bevorzugten Editor und tragen sie die dort angegebenen Attribute ein.

Nach erfolgreicher Installation können sie Rezervi über folgende URLs verwenden:

http://www.ihre-domain.com/IhrInstallationsverzeichnis/start.php

Dieser Startseite können Sie auch die gewünschte Sprache übergeben:

http://www.ihre-domain.com/IhrInstallationsverzeichnis/start.php?spracheNeu=de
http://www.ihre-domain.com/IhrInstallationsverzeichnis/start.php?spracheNeu=en
http://www.ihre-domain.com/IhrInstallationsverzeichnis/start.php?spracheNeu=fr

Den Zugang zur Webschnittstelle erhalten sie über:
http://www.ihre-domain.com/IhrInstallationsverzeichnis/webinterface/index.php

Bei Fragen können sie sich gerne jederzeit an uns wenden:

office@utilo.eu

http://www.utilo.eu

Danke für ihr Interesse an Rezervi!
