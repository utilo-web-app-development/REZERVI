Rezervi availability overview and guest database

Author: Christian Osterrieder UTILO

Installation of Rezervi

  1. Accept licence
    At first you have to accept the licence of Rezervi (see file "LICENCE.txt"). After that you can install and test the software.
  2. System requirements
    MYSQL datebase management system: MySQL >= 4.0.20-standard
    PHP version >= 4.2.3
    To check your PHP version you can execute the file     "doc/phpVersion.php" on your webserver.
    If your webserver is not implementing the system requirements, you can rent Rezervi. Please visit the site: http://www.rezervi.com/
  3. Unzip files
    Unzip the zip-archive resp. tar-archive on your webserver.
  4. Access to your MySQL database
    There is a folder "conf" in the unziped files. Inside you can find a file named "rdbmsConfig.php" (/conf/rdbmsConfig.php), edit this file. Therefore you have to use your username, your password and the name of your database. If you don't know this data contact your provider.
  5. Install and config the database
    In the folder "install" you can find the file "index.php". Execute this file to install Rezervi. Follow the instructions of this page. If there are any error messages, please check the access data to your database or contact UTILO.
  6. URLs
    • The main page of your availibility overview:
      http://your-domain/your-install-folder/start.php
    • You can add your favourite language to this page:
      http://your-domain/your-install-folder/start.php?sprache=de
      http://your-domain/your-install-folder/start.php?sprache=en
      http://your-domain/your-install-folder/start.php?sprache=fr
    • You can execute directly the search page:
      http://your-domain/your-install-folder/suche/index.php
    • You can also add your favourite language to this page:
      http://your-domain/your-install-folder/suche/index.php?sprache=de
      http://your-domain/your-install-folder/suche/index.php?sprache=en
      http://your-domain/your-install-folder/suche/index.php?sprache=fr
    • The access to the webinterface is with:
      http://your-domain/your-install-folder/webinterface/index.php
  7. Config Rezervi
    If you visit the webinterface for the first time, use the username 'test' and the password 'test'. Don't forget to change this after your first login.

    After that do the following configuration:
    1. adjustements -> languages
    2. change accommodation
    3. add rooms
    You can use Rezervi for your guests now.

  8. Test and registration
    You can test Rezervi at will. If you use it for your accommodation commercial your have to register you Rezervi version. After the registration you will get a bill. If you have paid you can use Rezervi.
    For the registration, see: www.rezervi.com

  9. Contact
    If you have problems with the installation or if you have found errors send us an email:
    office@utilo.eu
    We are looking forward to get your messages!
