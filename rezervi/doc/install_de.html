Autor: DI Christian Osterrieder UTILO

Installationsanweisungen zu Rezervi

  1. Lizenz akzeptieren
    Bevor Sie Rezervi installieren und Testen müssen Sie die Lizenz akzeptieren. Sie finden die Lizenz im Wurzelverzeichnis (Datei "LIZENZ.txt").
  2. Systemvoraussetzungen
    MYSQL-Datenbankmanagementsystem: MySQL >= 4.0.20-standard
    PHP-Version >= 4.2.3 Zur Überprüfung ihrer PHP-Version können Sie die Datei "doc/phpVersion.php" auf Ihrem Webserver ausführen.
    Falls Ihr Webserver diese Systemvoraussetzungen nicht erfüllt, können Sie Rezervi auch mieten. Für mehr Informationen besuchen Sie die Site: http://www.rezervi.com
  3. Dateien entpacken
    Entpacken Sie das erhaltene zip-Archiv bzw. tar-Archiv auf Ihrem Webserver.
  4. Zugang zur MySQL Datenbank festlegen
    Im Verzeichnis "conf" befindet sich die Datei "rdbmsConfig.php" (/conf/rdbmsConfig.php), modifizieren Sie diese Datei um den Zugang zu Ihrer MySQL Datenbank festzulegen. Dazu wird die URL zur Datenbank, der Datenbankname, der Benutzername und das Passwort benötigt. Falls Sie diese Daten nicht vorliegen haben, wenden Sie sich bitte an Ihren Provider.
  5. Datenbank installieren und Konfigurieren
    Sie finden im Verzeichnis "install" eine Datei "index.php". Nachdem Sie Rezervi auf Ihrem Webserver entpackt haben, können Sie dieses Script ausführen um Ihren Belegungsplan zu installieren. Führen Sie die Anweisungen dieser Seite aus, falls Fehlermeldungen erscheinen, prüfen Sie bitte die Zugangsdaten ihrer MySQL Datenbank oder wenden Sie sich an UTILO.
  6. URLs
    • Die Startseite Ihres Belegungsplanes:
      http://ihre-domain/ihr-Installationsverzeichnis/start.php
    • Dieser Startseite können Sie auch die gewünschte Sprache übergeben:
      http://ihre-domain/ihr-Installationsverzeichnis/start.php?sprache=de
      http://ihre-domain/ihr-Installationsverzeichnis/start.php?sprache=en
      http://ihre-domain/ihr-Installationsverzeichnis/start.php?sprache=fr
    • Sie können auch direkt die Suchfunktion aufrufen:
      http://ihre-domain/belegungsplanV01/suche/index.php
    • Auch dieser Seite kann die gewünschte Sprache übergeben werden:
      http://ihre-domain/ihr-Installationsverzeichnis/suche/index.php?sprache=de
      http://ihre-domain/ihr-Installationsverzeichnis/suche/index.php?sprache=en
      http://ihre-domain/ihr-Installationsverzeichnis/suche/index.php?sprache=fr
    • Den Zugang zur Webschnittstelle erhalten sie über:
      http://ihre-domain/ihr-Installationsverzeichnis/webinterface/index.php
  7. Rezervi konfigurieren
    Falls sie das erste mal die Webschnittstelle besuchen, benutzen sie den Benutzernamen 'test' und das Passwort 'test'. ändern sie danach diese Zugangsdaten in der Webschnittstelle.
    Führen sie danach folgende Konfigurationen durch:
    1. Einstellungen -> Sprachen festlegen
    2. Unterkunft ändern
    3. Zimmer anlegen
    Rezervi ist nun für ihre Gäste einsatzbereit.
  8. Test und Registrierung
    Sie können den Rezervi Belegungsplan und Gästedatenbank nach belieben Testen. Sollten Sie Rezervi allerdings kommerziell für Ihre Unterkunft einsetzen, ist eine Registrierung bei UTILO notwendig. Sie erhalten nach der Registrierung eine Rechnung und können danach Rezervi uneingeschränkt nutzen.
    Zur Registrierung: www.rezervi.com
  9. Kontakt
    Falls sie Probleme bei der Installation haben oder Fehler entdecken können sie uns gerne unter
    office@utilo.eu
    kontaktieren. Wir freuen uns auf ihre Nachricht!
